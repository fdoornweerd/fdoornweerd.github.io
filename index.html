Fully Connected Neural Network From Scratch

Created my own FCNN from the ground up with Python using NumPy to compute the linear algebra and multivariable calculus opperations. I trained my model on various datasets including the MNIST Digit Dataset, where I was able to achieve a 95% accuracy, and implemented optimization features such as dropout and momentum.

LinguaLink

My team was awarded 3rd place at WashU's 2023 Hackathon, where we created LinguaLink, a website that connects language learners. We developed an algorithm that connects similar users, a pathway for them to talk with matches, and a chatbot that converses with the user in the language they are learning. We utilized Angular for the frontend, Supabase for the database, and the OpenAI API for the chatbot.

Mandlebrot Set Viewer

Created an interactive viewer of the Mandlebrot Set that allows the user to zoom in on the infinite fractals created in the complex plane by an iterative function, with the color of the points depending on which iteration of the function it diverges.

Source Code

Maze Generator

I created an algorithm that randomly generates a solvable maze for any given size by using a queue to hold recursive calls of the function that creates the maze, so that it is evenly distributed amongst the space.
