Projects

Fully Connected Neural Network From Scratch

I created my own FCNN from the ground up with Python using NumPy to compute the linear algebra and multivariable calculus opperations. I trained my model on various datasets including the MNIST Digit Dataset, where I was able to achieve a 95% accuracy, and implemented optimization features such as dropout and momentum.

LinguaLink

My team was awarded 3rd place at WashU's 2023 Hackathon, where we created LinguaLink, a website that connects language learners. We developed an algorithm that connects similar users, a pathway for them to talk with matches, and a chatbot that converses with the user in the language they are learning. We utilized Angular for the frontend, Supabase for the database, and the OpenAI API for the chatbot.

Source Code

Maze Generator

I developed an algorithm in JavaScript that randomly generates a solvable maze for any given size by using a queue to hold the recursive calls of the function that creates the paths, allowing for it to be evenly distributed amongst the space.

Source Code

Mandlebrot Set Viewer

I built an interactive viewer of the Mandlebrot Set in JavaScript that allows the user to zoom in on the infinite fractals created in the complex plane by an iterative function, with the color of the points depending on which iteration of the function it diverges.

Source Code

Modeling COVID-19 Spread in a Linear Dynamical System

I worked on a team of three to analyze and predict trends in COVID-19 data by using MATLAB to create a SIRD model, finding the optimal matrix to properly represent the data. We ultimetly wrote a paper detailing our methods and findings.
